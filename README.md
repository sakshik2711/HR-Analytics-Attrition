# HR Analytics – Employee Attrition Prediction

## 📌 Project Overview

This project focuses on analyzing employee data to identify the key factors that lead to employee attrition and to build a machine learning model to predict which employees are likely to leave the organization

## 🎯 Objectives

* Identify factors affecting employee attrition
* Perform data analysis and visualization
* Build a predictive model using machine learning
* Provide actionable business insights

## 🛠 Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* Scikit-learn (Machine Learning)
* SHAP (Model Explainability)
* Power BI (Dashboard Visualization)
* Jupyter Notebook

## 📊 Key Analysis Performed

* Attrition distribution analysis
* Department-wise attrition trends
* Salary vs attrition comparison
* OverTime impact on attrition
* Years at company vs attrition

## 🤖 Machine Learning Model

* Model Used: Random Forest Classifier
* Data Split: 80% Training / 20% Testing
* Evaluation Metrics:

  * Accuracy
  * Confusion Matrix
  * Classification Report
## 🔍 Key Insights

* Employees working overtime are more likely to leave
* Lower salary is strongly linked with higher attrition
* Early tenure employees show higher attrition rates
* Work-life balance plays a significant role
## 📈 Dashboard (Power BI)

The interactive dashboard includes:

* Attrition Rate KPI
* Department-wise attrition
* Job Role analysis
* Salary trends
* Feature importance visualization
## 📥 Dataset

IBM HR Analytics Employee Attrition Dataset from Kaggle.
## 🚀 How to Run the Project

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Open Jupyter Notebook
4. Run all cells in the notebook
5. Open Power BI dashboard file
## 📌 Results

The model successfully predicts employee attrition with good accuracy and helps identify the most important factors affecting employee turnover.
